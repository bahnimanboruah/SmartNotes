# Notes Taking App

A full-stack notes taking web application built using **Vue.js** for the frontend and **Flask** for the backend.  
Users can create, edit, delete, and manage notes with a clean and responsive UI.

## Features
- Create, edit, and delete notes
- Responsive design
- RESTful API with Flask

- ## Tech Stack

### Frontend
- Vue.js
- Vue Router
- bootstrap

### Backend
- Flask
- Flask-RESTful
- Flask-CORS
- mysql
